## 컨벤션

### NED 좌표계 컨벤션

위치 벡터 형식: `[x_N, x_E, h]`

| 성분  | +             | 비고                                    |
| ----- | ------------- | --------------------------------------- |
| `x_N` | 북쪽 (North)  | 표준                                    |
| `x_E` | 동쪽 (East)   | 표준                                    |
| `h`   | **위쪽 (Up)** | 표준 NED의 z_down을 반전 → `h = -z_NED` |

이 컨벤션을 프로젝트 전체(경로점, 상태, 플래너 API)에서 일관되게 적용해야 합니다.

- [`base_dynamics.py`](dynamics/base_dynamics.py) 주석: `"h = -z_NED, 양수 = 위쪽"`
- [`geodetic.py`](utils/geodetic.py): `h = alt - ref_alt` (고도 증가 = h 증가)

### 각도 컨벤션

| 변수        | 정의       | +                         |
| ----------- | ---------- | ------------------------- |
| `chi` (χ)   | 방위각     | True North 기준, 시계방향 |
| `gamma` (γ) | 비행경로각 | 상승 방향                 |
| `phi` (φ)   | 뱅크각     | 우선회 방향               |

### 바람 벡터: `[w_N, w_E, w_D]`

`w_D > 0` = **아래 방향** (표준 NED), 우리 NED 컨밴션과 z방향 반대.

### Body-frame 가속도: `[a_x, a_y, a_z]`

- `a_x`: 전방 (Forward) +
- `a_y`: 우측 (Right) +
- `a_z`: 하방 (Down) + (우리 NED z 컨벤션과 동일)

### 컨벤션 취급 주의사항

표준 NED에서는 Z축이 Down(+)이지만, 이 프로젝트는 **고도 `h`와 Up(+)으로 통일**했습니다.(실제 고도값 그대로 사용 가능.)
내부적으로 `z_NED = -h`로 변환해서 사용합니다.
바람의 `w_D`만 예외적으로 표준 NED를 그대로 씀에 주의합시다.
